Machine Learning-Based Customer Churn Prediction
Overview:
This project focuses on predicting customer churn using machine learning classification models. Customer churn means identifying customers who are likely to stop using a service. The project uses the Telco Customer Churn dataset and applies data preprocessing, model training, and evaluation techniques to compare different machine learning models.

Objective:
The main objective of this project is to build a machine learning model that can predict whether a customer is likely to churn or not based on customer information and service usage data.

Dataset:
The dataset used in this project is the Telco Customer Churn dataset. It contains customer-related information such as contract type, internet service, payment method, tenure, monthly charges, and churn status.

Project Workflow
Imported the required Python libraries.
Loaded and explored the dataset.
Cleaned and preprocessed the data.
Encoded categorical features.
Split the data into training and testing sets.
Trained multiple machine learning models.
Evaluated model performance using accuracy, confusion matrix, ROC curve, and AUC score.
Compared the results to identify the best-performing model.
Results
The models were evaluated based on classification performance. Logistic Regression performed well compared to the other models and gave a reliable result for churn prediction.
